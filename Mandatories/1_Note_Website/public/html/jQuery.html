Brug af jQuery i et projekt

Her bliver beskrevet hvordan man kan anvende jQuery til at gøre html-sider mere intuitive, samt gøre brugeroplevelsen bedre.

jQuery er et platformsuafhængigt JavaScript-bibliotek der har til formål at gøre det nemmere at lave programmer til webbrowsere.

Hvordan virker jQuery?

jQuery virker ved at kalde forskellige 'selectors', for derigennem at lave ændringer på websiden.

Selectors kan deles op i tre forskellige kategorier:

  • tag
  • class
  • id

    • Tags er html tags såsom: head, body, div, h1-h6, p, etc. - og kan bruges til at sørge for at alle elementer af et givent tag får en række ekstra egenskaber.

    Classes er noget man kan tilføje til sine html tags, som kan sørge for at man selv kan bestemme hvilke elementer, der skal have nogle ekstra egenskaber tilføjet på tværs af tags. Classes bliver markeret med et . punktum foran, fx ".images"

    Id'er er også noget man kan tilføje til et html tag, men adskiller sig fra classes ved at de er unikke - man kan altså ikke tilføje et id til flere end et element. Id's bliver markeret med et # hashtag foran, fx "#image".

    Gennem en af disse tre metoder, kan man bruge jQuery til at tilføje egenskaber til sine html-elementer - hvilket gøres efter en html side er blevet loaded.

    Antallet af forskellige muligheder for særlige egenskaber og effekter er mange, og følgende liste giver et overblik på de mange muligheder man har for at ændre i indholdet på html-siden.

    jQuery kan bl.a. bruges til:

  • HTML/DOM manipulationer
  • CSS manipulationer
  • HTML event metoder
  • Effekter and animationer
  • AJAX
  • Utilities

    • Kom igang med jQuery

    For at bruge jQuery's bibliotek, skal det inkluderes i projektet - dette kan gøres på to forskellige måder.
    Man kan vælge at have JS koden nederst i sit dokument som internt, eller man kan vælge at oprette et eksternt javascript dokument, og linke det til html-siden.

    Intern brug af jQuery

    Ved intern brug af jQuery, skal man nederst i html dokumentet - lige inden body'en slutter - lave et script tag. Derinde kan man skrive javascript kode, og lave sine jQuery funktioner. Se på billedet nedenunder, for at se hvor script tagget skal være.

    Ekstern brug af jQuery

    Ved ekstern brug af jQuery skal der oprettes en ny js fil, den skal lægges inde i public mappen for at den er tilgængelig. I dette eksempel er der lavet en mappe kaldet 'js', og inde i denne mappe lægges js filen.
